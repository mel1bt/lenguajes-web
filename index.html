Algunos lenguajes de la web

HTML+CSS+JAVASCRIPT

  • HTML 5

    HTML 5(HyperText Markup Languaje, version 5)es la quinta revision importante del lenguaje básico de la World Wide Web.HTML.

    Usa la extención .html

  • CSS 3

    Hoja de estilos en cascada o CSS(siglas en inglés de cascading style sheets)es un lenguaje usado para definir y crear la presentacion de un documento estructurado escrito en HTML. Usa la extención .css

  • JavaSript

    JavaSript Abreviado comunmente "JS" es un lenguaje de programación utilizado principalmente en el lado del cliente (client-side), implementando como parte de un navegador web permitiendo mejoras en la intefaz de usuario y páginas web dinámicas. usa la extensión .js
